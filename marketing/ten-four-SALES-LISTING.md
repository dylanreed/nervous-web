# TenFour - Discord Ticketing Bot

## The Simple Support Solution for Discord Communities

**Set up in 60 seconds. No web dashboard required. Everything happens in Discord.**

---

### Why TenFour?

Other ticketing bots make you jump through hoops. Web dashboards. Complicated configs. 15-minute setup processes.

**TenFour is different.** One command. Three questions. Done.

---

### Features

**Dead-Simple Setup**
- `/setup_ten_four` and you're live
- Interactive wizard guides you through everything
- No external website needed

**Modern Discord Experience**
- Buttons, dropdowns, and modals (not clunky reactions)
- Private ticket channels for each request
- Clean, professional embeds

**Everything Your Team Needs**
- Claim tickets to show ownership
- Add users to bring in specialists
- Category routing for different teams
- Configurable intake forms

**Never Lose a Conversation**
- Beautiful HTML transcripts
- Automatic logging to your staff channel
- Full message history with attachments

**Keep Tickets Moving**
- Inactivity reminders (configurable timing)
- Satisfaction ratings after close
- Activity tracking on every message

### Commands

| Command | Description |
|---------|-------------|
| `/setup_ten_four` | Configure the bot for your server |
| `/ticket` | Create a support ticket |
| `/ping` | Check if bot is online |

---

### Getting Started

1. **Invite** TenFour to your server
2. **Run** `/setup_ten_four`
3. **Pick** your panel channel, support role, and log channel
4. **Done** - Users can now create tickets!
