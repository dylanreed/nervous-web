Setup Guide

Step 1: Install the App

Install Merch Table from the Shopify App Store. After installation, you'll see the app dashboard.

Step 2: Connect Your Discord Server

  1. On the Dashboard tab, click the "Invite Discord Bot" link
  2. Select your Discord server and authorize the bot
  3. Copy the Connection Code shown on the dashboard
  4. In your Discord server, type /connect
  5. Enter the connection code when prompted
  6. Wait a few seconds - the dashboard will show "Connected"

Step 3: Configure Discord Roles

  1. Go to the Discord Roles tab in Shopify
  2. Toggle ON the roles you want to sync (e.g., VIP, Member, Subscriber)
  3. Toggle OFF admin/moderator roles you don't want synced
  4. Click Save Configuration

When customers link their accounts, these roles become Shopify customer tags (e.g., discord_vip).

Step 4: Customers Link Their Accounts

Your community members link their Discord to Shopify by:

  1. Running /link_shopify in your Discord server
  2. Entering their email address (must match their Shopify account)
  3. Clicking the verification link sent to their email
  4. Logging into Shopify to complete the link

Their Discord roles are now synced as Shopify customer tags!

Step 5: Set Up Role-Based Discounts (Optional)

  1. Go to the Discounts tab
  2. Click + Add Discount
  3. Select a Discord role tag (e.g., discord_vip)
  4. Set the discount type (percentage or fixed amount)
  5. Click Create Discount

Customers with that role automatically get the discount at checkout!

Step 6: Set Up VIP Products (Optional)

Create an exclusive section for Discord members:

  1. Go to the Product Visibility tab
  2. Fill in your Discord invite URL, page title, and description
  3. Click Save Settings
  4. Click + Add Product to add exclusive items
  5. Select a product and choose which Discord roles can see it
  6. Add the VIP URL to your store's navigation: yourstore.com/apps/merch-table/vip

Step 7: Enable Cart Validation (Optional)

Prevent customers without required roles from purchasing VIP products:

  1. In Shopify admin, go to Online Store > Themes > Customize
  2. Click App embeds in the left sidebar
  3. Find "Cart Validation" and toggle it ON
  4. Click Save

Commands Reference

Admin Commands

These commands require Discord Administrator permission.

/connect

Connect your Discord server to a Shopify store.

  • code (required) - Connection code from Shopify app (format: CONNECT-ABC123)

Usage:

  1. Install Merch Table in your Shopify store
  2. Copy the connection code from the Shopify admin dashboard
  3. Run /connect code:CONNECT-ABC123 in your Discord server

Note: Each Discord server can only connect to one Shopify store.

/shopify_info

View connection information for your server.

Displays:

  • Connected Shopify store domain
  • Total number of linked users
  • New links in the past 24 hours
  • Connection timestamp

User Commands

These commands are available to all server members.

/link_shopify

Link your Discord account to a Shopify customer account.

Usage:

  1. Run /link_shopify
  2. Enter the email address associated with your Shopify account in the modal
  3. Your Discord roles will automatically sync as Shopify customer tags

Notes:

  • Rate limited to once every 5 minutes
  • You must have an existing Shopify customer account with the connected store
  • Your current Discord roles are sent to Shopify as tags (e.g., role "VIP" becomes tag discord_vip)

/unlink_shopify

Remove the link between your Discord and Shopify accounts.

What happens:

  • Your Discord account is unlinked from Shopify
  • Discord role tags are removed from your Shopify customer profile
  • You will lose access to any role-gated products or discounts

/shopify_status

Check your current link status with Shopify.

Displays:

  • Whether you're linked to a Shopify account
  • Your current Discord roles being tracked
  • Shopify tags that are synced
  • Last sync timestamp

Command Summary

Command Permission Description
/connect Admin Connect server to Shopify store
/shopify_info Admin View server connection info
/link_shopify Everyone Link Discord to Shopify account
/unlink_shopify Everyone Remove Discord-Shopify link
/shopify_status Everyone Check your link status

Permissions Guide

Discord Command Permissions

User Type Access Level
Discord Admin Can connect server, view connection info
Discord Member Can link/unlink account, check status
Shopify Admin Full dashboard access, customer management

Shopify Admin Access

The Shopify admin dashboard is protected by Shopify session validation.

What admins can do:

  • Connect/disconnect Discord servers
  • View and manage linked customers
  • Configure role-based discounts
  • Set up product visibility rules
  • Manage VIP section settings
  • Export customer data
  • Force-sync customer tags

Role-Based Features

Discord Role Sync

When a user links their account:

  1. Their Discord roles are captured
  2. Roles are converted to Shopify tags with discord_ prefix
  3. Example: Role "VIP Members" becomes tag discord_vip_members

Note: @everyone is always excluded from sync. Admins can exclude additional roles from the Shopify dashboard.

Data Isolation

All data is scoped to individual shops:

  • Discord servers can only connect to one Shopify store
  • Shopify stores can only connect to one Discord server
  • Customer links are isolated per shop

Rate Limiting

Action Cooldown
/link_shopify command 5 minutes per user

Features

Role-Based Discounts

Discounts can be configured to automatically apply based on Discord roles:

Setting Description
Role Which Discord role triggers the discount
Discount Code The Shopify discount to apply
Auto-Apply Whether to apply automatically at checkout

Product Visibility

Products can be restricted to users with specific Discord roles:

  • Uses Shopify product metafields
  • Cart validation checks for required tags
  • Products hidden from users without required role

Troubleshooting

Bot not responding to commands?

  • Make sure the bot has been invited and has proper permissions
  • Try /shopify_info to verify the server is connected

Customer's roles not syncing?

  • Check that the role is toggled ON in the Discord Roles tab
  • Have the customer run /shopify_status to trigger a sync

VIP page not loading?

  • Verify App Proxy is configured correctly in Partner Dashboard
  • Check that the proxy URL is correct
